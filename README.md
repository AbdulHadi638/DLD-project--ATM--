# DLD-project-
Designed a simplified ATM system using digital logic design and Finite State Machines (FSMs). Implemented PIN verification, transaction processing, and balance display using logic gates, flip-flops, multiplexers, and 7-segment displays. Simulated core ATM operations with automated state transitions.

1: Introduction
 This project aims to design and implement a simplified ATM system using digital logic components such as logic gates, flip-flops, counters, and multiplexers. The system will simulate basic ATM operations like PIN verification and cash withdrawal.
2: Mechanism:
The system will be designed using concepts of Finite State Machine’s (FSM’s) and will contain the following main areas;
i) Power ON 
ii) Enter PIN
iii) PIN verification
iv) Withdrawal/Deposit Amount Entered
v) Balance Displayed ( Increased or Decreased)
3: Components:
The Components required will be
i) Logic Gates (AND, OR, NOT)
ii) Flip-Flops 
iii) Multiplexers
iv) Counters  ( OPTIONAL ) 
v) 7-Segment Display
vi) BCD to 7-Segment Decoder (7447)
vii) Push Buttons / Switches
viii) Clock Pulse Generator (555 Timer)

4: Working:
The ATM system will use a Finite State Machine (FSM) which will be implemented using flip-flops and logic circuits. Initially, the system remains in the idle state and waits for user input through switches. The user then enters a 4-bit PIN using switches, which is verified using combinational logic gates (AND, OR, NOT). If the entered PIN matches the correct PIN ( predefined), the system moves  to the next stage,, otherwise, an error is generated using LEDs. If PIN is correct, the user selects a withdrawal amount using push buttons, and a multiplexer is used to process the selected option. The corresponding output is then displayed using a 7-segment display which is used through a BCD to 7-segment decoder. A clock signal generated by a 555 timer ensures proper working of state transitions and sequential operations. After completing the transaction, the system resets back to the idle state, ready for the next operation.
